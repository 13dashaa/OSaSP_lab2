## Установка и запуск

1. **Перейдите в каталог проекта:**

    ```bash
    cd 'Макушинская Д.В./lab01'
    ```

2. **Соберите проект с помощью make:**

    ```bash
    make 
    ```
   
3. **Установите переменную окружения CHILD_PATH**

    ```bash
    export CHILD_PATH=./child
    ```
3. **Запуск программы:**

    ```bash
    ./parent child_path
    ```
